# 개발환경
**프로젝트 기간** : 2023.03 ~ 2023

**개발 웹서버** : Apache Tomcat 9.0

**개발 툴** : 전자정부프레임워크 3.10, MYSQL

**사용 언어** : Java(JDK 1.8), SQL, JSP & Servlet, HTML/CSS, JavaScript, Jquery

로그인
